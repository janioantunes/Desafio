Button

data-corners: Aplica cantos arredondados aos botões com o valor true e cantos quadrados com o valor false.

false true

data-icon: Aplica diferentes ícones aos botões. Os exemplos abaixo demonstram os valores possíveis e os ícones que são aplicados com os valores:

home delete plus arrow-u arrow-d check gear grid star custom arrow-r arrow-l minus refresh forward back alert info search

data-iconpos: Utilizado como complemento do data-icon. Define a posição em que o ícone aparecerá nos botões. Os exemplos abaixo demonstram os valores possíveis e os ícones que são aplicados com os valores:

left right top bottom notext

data-iconshadow: Aplica sombra tema para o ícone do botão se o valor definido for true.

true false

data-inline: Se definido como true, fará o botão ter a largura determinada pelo texto do botão. Por padrão, o valor é false para que o botão tenha a largura da página, independente do tamanho do deu conteúdo.

true false

data-mini: Se o valor definido for true, irá mostrar uma versão mais compacta do botão que usa menos altura vertical.

true false

data-shadow: Aplica o estilo de sombra para o botão se o valor for definido como true.

true false

data-theme: Define o esquema de cores para todas as instâncias do botão. Ele aceita uma única letra entre a e z, que mapeia para uma das amostras incluídas no seu tema. Por padrão, ele herdará a mesma amostra de cor como seu contêiner pai, se não for explicitamente definido.

a b c d e

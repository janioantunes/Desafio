data-corners: Define cantos arredondados para os popups com o atributo true e cantos quadrados com o atributo false.

true

Cantos arredondados.

false

Cantos quadrados.

data-dismissible: O atributo true define que, se clicar fora do popup ou pressionar ESC enquanto o popup estiver aberto, vai fecha-lo. O atributo false permite fechar o popup somente ao pressionar o botão "voltar".

true

Pressine "ESC" para sair.

false

Pressine "voltar" para sair.

data-history: O atributo true define se o popup cria um item de histórico do navegador quando se abre. Se definido como false, não vai criar um item de histórico e não vai permitir ser fechado ao pressionar o botão "voltar".

true

Pressine "ESC" para sair.

false

Pressine "voltar" para sair.

data-position-to: O parametro origin indica que o popup deve ser centrado sobre o link que o abre. O parametro window indica que ele deve ser centralizado na janela do navegador. O parametro selector será centrado sobre o primeiro elemento retornado pelo seletor, se esse elemento é visível. Caso contrário, ele será centralizado na janela do navegador.

origin

Pressine "ESC" para sair.

Pressine "ESC" para sair.

data-tolerance: Define a distância mínima a partir da borda da janela para a borda correspondente do popup. Por padrão, os valores "30,15,30,15" serão utilizados para a distância do topo (top), direita (right), inferior(bottom) e esquerda (left) da janela, respectivamente.

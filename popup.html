Popup

data-corners: Define cantos arredondados para os popups com o atributo true e cantos quadrados com o atributo false.

true

Cantos arredondados.

false

Cantos quadrados.

data-dismissible: O atributo true define que, se clicar fora do popup ou pressionar ESC enquanto o popup estiver aberto, vai fecha-lo. O atributo false permite fechar o popup somente ao pressionar o botão "voltar".

true

Pressine "ESC" para sair.

false

Pressine "voltar" para sair.

data-history: O atributo true define se o popup cria um item de histórico do navegador quando se abre. Se definido como false, não vai criar um item de histórico e não vai permitir ser fechado ao pressionar o botão "voltar".

true

Pressine "ESC" para sair.

false

Pressine "voltar" para sair.

data-overlay-theme swatch letter (a-z) - Default "null" (transparent background) data-position-to origin | window | selector - "origin" indicates that the popup is to be centered over the link that opens it. "window" indicates that it is to be centered in the browser window. If you give a jQuery selector, it will be centered over the first element returned by the selector, if that element is visible. Otherwise, it will be centered in the browser window. data-shadow true | false data-theme swatch letter (a-z) | none - Default inherited, "none" sets the popup to transparent data-tolerance 30,15,30,15 - Distance from the edges of the window (top, right, bottom, left) data-transition none | transition - The transition to use when making the popup appear/disappear. Anchor with a data-rel="popup" opens the popup data-position-to origin - Centers the popup over the link that opens it jQuery selector - Centers the popup over the specified element window - Centers the popup in the window data-rel popup - For opening a popup data-transition fade | flip | flow | pop | slide | slidedown | slidefade | slideup | turn | none - The transition to be used when showing/hiding the popup

Dialog

data-close-btn: Posiciona o botão de fechar a caixa de diálogo conforme valores demonstrados abaixo:

left right none

data-close-btn-text: Permite alterar o texto padrão (close)do botão fechar para outro texto personalizado.

texto

data-corners: Define cantos arredondados para a caixa de diálogo com o valor true e cantos quadrados com o valor false.

true false

data-overlay-theme: Diálogos parecem estar flutuando acima de uma camada de sobreposição. Essa sobreposição adota o tema do conteúdo por padrão, mas com o data-overlay-theme valores entre a e z podem ser utilizados para definir outros temas para a camada de sobreposição.

a b c d e

data-theme: Define o esquema de cores para a caixa de diálogo. Ele aceita uma única letra entre a e z, que mapeia para uma das amostras incluídas no seu tema. Por padrão, ele herdará a mesma amostra de cor como seu contêiner pai, se não for explicitamente definido.

a b c d e

data-title: Permite definir um titulo personalizado para a janela da caixa de diálogo, que aparecerá na aba/janela do browser.

texto
